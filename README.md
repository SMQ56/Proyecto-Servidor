# Sueños Valenti – DWES Project

Server-side web application developed using **Node.js**, **Express**, and **EJS**.

## Technologies Used
- Node.js  
- Express  
- EJS  
- express-session  
- cookie-parser  

## Project Structure
- **/views** → EJS templates  
- **/routes** → Application routes  
- **/public** → Static files (CSS)  
- **/data** → JSON data files
